    Vrijer leuker en makkelijker

    Welkom bij de Cloud FAQ. Deze app is gemaakt om het werk vrijer leuker en makkelijker te maken.

    De inhoud is niet officieel geaccordeerd en dat zijn de antwoorden die ik geef bij hulpvragen ook niet. Gebruik dit in je voordeel, tenzij je beter weet.

    Sluit deze app af indien je niet volledig akkoord gaat met de Cloud FAQ Voorwaarden, hou het eenvoudig en ga verder met de fijne dag.

    Heb je het FAQ onder de knie?

    Scoor je al met de minste moeite de hoogste tevredenheid? Dan is deze app speciaal voor jou! Gebruik dit om oplossingen, workarounds of extra informatie te delen om meer promotors te scoren met het team.

    Autorisatie

    Autorisatie is vereist om deze app te gebruiken en is wederzijds. Gebruik is kosteloos in ruil voor analytics. Verlaat deze app onmiddellijk om te stoppen.

    Uitleg

    Solution, Workaround of Extra informatie:

    package main

import "github.com/bep/kittn/auth"

func main() {
	api := auth.Authorize("meowmeowmeow")

	// Just to make it compile
	_ = api
}

    Aan deze kant (de linkerkant) lees je de klantvraag of de context. Aan de andere kant (de rechterkant) lees je de bijbehorende oplossing, workaround of extra informatie.

    Zoekfunctie

    Gebruik de zoekfunctie om te zoeken op klantvraag, product of context. De zoekfunctie zit in het menu. Klik op NAV indien het menu verborgen is.

    Get All Kittens

    package main

import "github.com/bep/kittn/auth"

func main() {
	api := auth.Authorize("meowmeowmeow")

	_ = api.GetKittens()
}

    require 'kittn'

api = Kittn::APIClient.authorize!('meowmeowmeow')
api.kittens.get

    import kittn

api = kittn.authorize('meowmeowmeow')
api.kittens.get()

    curl "http://example.com/api/kittens"
  -H "Authorization: meowmeowmeow"

    const kittn = require('kittn');

let api = kittn.authorize('meowmeowmeow');
let kittens = api.kittens.get();

    This endpoint retrieves all kittens.

    HTTP Request

    GET http://example.com/api/kittens

    Query Parameters

    Parameter Default Description
    include_cats false If set to true, the result will also include cats.
    available true If set to false, the result will include kittens that have already been adopted.
    Remember — a happy kitten is an authenticated kitten!

    Errors

    Noteer de foutcode en zoek de verklaring hier op!

    Uitleg bij een foutmelding in Cloud FAQ:

    Foutmelding Uitleg
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
